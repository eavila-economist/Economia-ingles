## Estudo

### Cursos

* [MIT OCW Economics](http://ocw.mit.edu/courses/economics/) - Mais de 100 cursos cobrindo todas as principais áreas da economia. Os cursos incluem pré-requisitos, livros recomendados, slides de palestras e tarefas. Programas de graduação e pós-graduação.
* [edX Economics](https://www.edx.org/course/subject/economics-finance) - Tópicos introdutórios.
* [Khan Academy: Economics](https://www.khanacademy.org/economics-finance-domain) - Tópicos elementares.

### Materiais Úteis

* [Equações Fundacionais da Economia](http://economics.stackexchange.com/questions/35/what-are-the-foundational-equations-of-economics) - Essas equações mostram os princípios da economía. Os livros didáticos de pós-graduação colocam essas equações em contexto.
* [Painel de Especialistas Econômicos do IGM](http://www.igmchicago.org/igm-economic-experts-panel) - Os principais economistas refletem sobre questões relacionadas a políticas. Algumas respostas contêm detalhes úteis.
* [AEA Resources for Economists](http://rfe.org/) - Uma lista de links úteis mantida pela American Economic Association.

##Pesquisa

### Portais

* [RePEc](http://repec.org/) - Web services para pesquisadores econômicos: bibliografia, blogs, novos documentos de trabalho, software.
* [Federal Reserve Bank of St. Louis: Web Services](https://research.stlouisfed.org/publications) - Serviços e informações da Federal Reserve Bank of St. Louis.

### Artigos e Documentos de Trabalho

* [IDEAS RePEc](https://ideas.repec.org/) - O maior banco de dados de publicações de economia (2.000.000 itens). Pesquisar em artigos é mais fácil com o Google: ``site:ideas.repec.org <termo de pesquisa>``.
* [NBER](http://nber.org/papers.html) - Documentos de trabalho de grandes pesquisadores. Muitos desses artigos são publicados em um periódico revisado por pares.
* [SSRN Economics](http://ssrn.com/en/index.cfm/ern/) - Documentos de trabalho, sem publicações em periódicos.
* [Google Scholar](http://scholar.google.com/) - Pesquisa de literatura acadêmica em geral. 

### Dados

Conjuntos de dados

* [FRED2](https://fred.stlouisfed.org/) - 380.000 séries de 80 fontes. Suporta plugins para importar dados para Excel, Stata, R e outros. 
* [Dados do Banco Mundial](http://data.worldbank.org/) - Série temporal macro internacional. Tenha plugins de importação de dados.
* [Dados do IMF](https://data.imf.org/?sk=388dfa60-1d26-4ade-b505-a05a558d9a42) - A referência padrão para dados macro.
* [Quandl](https://www.quandl.com/) - dados financeiros e econômicos agregados de várias fontes. Alguns fornecedores de dados vendem seus dados por meio desse serviço. Boa integração com software estatístico.
* [MEDevEcon](https://sites.google.com/site/medevecon/devecondata) - Dados relacionados à economia do desenvolvimento.
* [Economia monetária: fontes de dados](https://sites.google.com/site/davesmant/various/data-sets-sources) - Visão geral das fontes de dados macro.
* [OFFSTATS](http://www.offstats.auckland.ac.nz/) - Links para fontes de dados oficiais por país e assunto.

Procurar

* [Mecanismo de pesquisa de conjunto de dados](https://cse.google.com/cse/publicurl?cx=002720237717066476899:v2wv26idk7m) - pesquisa com base no Google em mais de 200 fontes de dados, incluindo as mencionadas aqui. Você pode usar o Google [operadores de pesquisa](http://www.googleguide.com/advanced_operators_reference.html) aqui.
* [StackExchange Open Data](http://opendata.stackexchange.com/) - Se você não encontrou os dados que estava procurando, pode perguntar aqui.
* [Reddit /r/datasets](https://www.reddit.com/r/datasets) - Mais um lugar para solicitar conjuntos de dados.

### Programas

Escrita

* [LaTeX](https://www.latex-project.org/) - Economistas escrevem em LaTeX porque ele lida com matemática e referências melhor do que Word ou LibreOffice. Se você escreve regularmente, vale a pena aprender LaTeX.
  - [Beamer](https://www.ctan.org/pkg/beamer?lang=en) - Uma aula de LaTeX para apresentações.
  - [TikZ](http://www.texample.net/tikz/examples/) - Uma extensão para desenhar gráficos. Um [como fazer](http://cremeronline.com/LaTeX/minimaltikz.pdf) e um [manual](http://www.texample.net/media/pgf/builds/pgfmanualCVS2012-11-04.pdf ).
* [LyX](http://www.lyx.org/) - Um editor simples e gratuito para LaTeX.
* [Zotero](https://www.zotero.org/) - Gerenciamento de bibliografia. Instale também (a) o plugin do navegador Zotero para importar artigos do RePEc para sua biblioteca; (b) Plugin Zotero-LyX para citar literatura facilmente.


Informática

* [IPython](http://ipython.org/) - Um ambiente baseado em Python. A análise econométrica é feita com pacotes gratuitos: statsmodels, SciPy, NumPy, pandas.
* [RStudio](https://www.rstudio.com/) - Um ambiente baseado em R. R é a linguagem padrão entre os estatísticos, portanto, os repositórios R geralmente contêm bibliotecas especializadas não disponíveis em outras linguagens.
* [Matlab](http://www.mathworks.com/products/matlab/) - Um padrão da indústria para modelagem e otimização numérica em economia. Alternativas gratuitas:
  - [Octave](https://www.gnu.org/software/octave/)
  - [Julia](https://julialang.org/) - Linguagem de programação dinâmica de alto nível projetada para atender às necessidades de análise numérica de alto desempenho e ciência computacional.
* [Mathematica](https://www.wolfram.com/mathematica/) - Cálculos simbólicos. alternativa gratuita
  - [Sage](http://www.sagemath.org/)

* Blogs - A forma mais popular de auto-expressão entre os economistas. Os principais blogs:
  - [EconAcademics.org](http://econacademics.org/)
  - [Economist's View](http://economistsview.typepad.com/) - Mark Thoma
  - [Grasping Reality](http://delong.typepad.com/) - Brad DeLong
  - [Marginal Revolution](http://marginalrevolution.com/) - Tyler Cowen e Alex Tabarrok
  - [Deveconodata Blogspot](http://devecondata.blogspot.com/) - Conjuntos de dados de economia de desenvolvimento. Atualizado regularmente.
  - [VOX CEPR](http://www.voxeu.org/) - Membros do CEPR
  - [Top 100 Economics Blogs](https://www.intelligenteconomist.com/top-economics-blogs-2016/) - Links para blogs de economia populares, com breves descrições.
* [Pesquisa de blogs de economia](https://cse.google.com/cse/publicurl?cx=002720237717066476899:i_zfef5fawa) - Um serviço de pesquisa baseado no Google para os blogs mencionados acima.
* [AEA Blog Directory](https://www.aeaweb.org/rfe/showCat.php?cat_id=96) - A lista dos principais blogs econômicos.
* [StackExchange Economics](http://economics.stackexchange.com/) - Um site de perguntas e respostas onde você pode fazer e responder perguntas.
* Reddit - Um agregador de notícias popular. Tem muitas seções relacionadas à economia, por exemplo:
  - [/r/GoodEconomics/](https://www.reddit.com/r/GoodEconomics/) - Peças selecionadas sobre questões econômicas.
  - [/r/EconPapers/](https://www.reddit.com/r/EconPapers/) - Discutindo documentos econômicos.
* Discord - Uma plataforma de bate-papo popular
  - [Economia Acadêmica](https://discord.gg/n9V7hjNtPV) - Uma comunidade com salas para discutir economia e ajudar os membros com exercícios
